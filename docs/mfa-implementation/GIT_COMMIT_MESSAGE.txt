## Overview
Complete overhaul of MFA management in Profile > Security to use Auth0 My Account API for user self-service authentication method management.

⚠️ **Status:** Implementation complete but not yet active
- Waiting for Auth0 to enable My Account API feature flags via ESD ticket
- Legacy MFA system backed up and can be restored if needed
- See MFA_ROLLBACK.md for rollback instructions

## New Features
- Consolidated MFA management UI with "Enrolled Methods" and "Available Factors"
- My Account API integration for CRUD operations on authentication methods
- Step-up authentication enforcement for all sensitive operations
- Unified RESTful API endpoints (/api/mfa/methods, /api/mfa/factors)
- Support for 5 MFA factor types: SMS, TOTP, WebAuthn-Roaming, WebAuthn-Platform, Email
- Clean card-based UI with better UX
- In-app enrollment (no Guardian page redirects)
- Individual method removal and full MFA reset
